
## Running the app:

```bash

$ yarn run start

$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Tests:

```bash

$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
